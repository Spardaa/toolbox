JSONPath Syntax Reference

$ Root object/element
@ Current object/element
. Child member
.. Recursive descent
* Wildcard - all objects/elements
[start:end] Array slice
[n] Array index (0-based)
[n,m,...] Multiple array indices
[?(expr)] Filter expression
[\'name\'] Bracket notation for key
