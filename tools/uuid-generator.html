ℹ️ About UUID Versions

UUID v1 (Time-based)

Generated using the current timestamp and machine MAC address. Provides uniqueness but reveals creation time and machine identity.

UUID v4 (Random)

Generated using random numbers. Most common version. Provides 122 bits of randomness with extremely low collision probability.

UUID v5 (Namespace)

Generated using SHA-1 hash of a namespace and name. Same namespace + name always produces the same UUID. Useful for deterministic IDs.

Common Use Cases

Database primary keys, session IDs, transaction IDs, distributed systems identifiers, and any scenario requiring unique identifiers.
